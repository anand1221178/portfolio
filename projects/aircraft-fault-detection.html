Project Overview

This project implements a fault detection system for aircraft engines using a Dynamic Bayesian Network (DBN) integrated with a Markov Random Field (MRF). The system analyzes multivariate sensor data from simulated aircraft engine scenarios to distinguish between actual mechanical degradation and sensor failures. The solution achieves 85.1% accuracy in fault classification by modeling temporal dependencies and smoothing noisy sensor data.

Key Features

Dynamic Bayesian Network

Models temporal transitions of engine subsystem health and sensor observations.

MRF Smoothing

Improves robustness to inconsistent vibration readings from colocated sensors.

Sensor Health Modeling

Enables robustness to sensor drift and failure, a critical aspect of real-world scenarios.

Simulated Scenarios

Evaluated on five realistic operational scenarios, including normal operation, oil leaks, bearing wear, and sensor failures.

Technical Implementation

The core of the project is the DBN, which captures temporal dependencies, and the MRF, which smooths inconsistent vibration readings. The system is designed to be robust to sensor drift and failure.

1. Simulated Scenario Generation

The system was tested on five realistic operational scenarios over 1800 timesteps:

  • Normal Operation: Baseline for a healthy engine.
  • Oil Leak: Simulates a progressive drop in oil pressure.
  • Bearing Wear: Models increasing vibration amplitude over time.
  • EGT Sensor Failure: Introduces a fixed bias in exhaust gas temperature readings.
  • Vibration Sensor Failure: Simulates random noise spikes in vibration sensors.

Challenges and Solutions

Vibration Noise

Challenge: High levels of noise in vibration sensor readings made it difficult to distinguish between normal variations and actual bearing wear faults.

Solution: Implemented MRF smoothing to enforce temporal consistency, reducing false positives by 15% while maintaining detection sensitivity.

Sensor vs Mechanical Faults

Challenge: Distinguishing between actual mechanical degradation and sensor failures that mimic fault signatures.

Solution: The DBN was structured to model sensor health, allowing it to differentiate between true faults and sensor malfunctions. This resulted in an 85.1% accuracy across all 5 scenarios.

Key Insights

  • Quantile binning (5 bins) outperformed k-means and uniform discretization.
  • MRF smoothing with λ=0.7 was optimal for vibration channels, reducing false positives by 15%.
  • The DBN's temporal modeling successfully captured progressive degradation in oil leaks and bearing wear.
  • A 9-sensor subset, selected via correlation analysis, matched the performance of the full sensor suite.

Conclusion

This project demonstrates the effectiveness of probabilistic graphical models in handling complex, noisy sensor data for fault detection. The combination of a Dynamic Bayesian Network with MRF smoothing achieved 85.1% accuracy across 5 simulated scenarios, successfully distinguishing mechanical faults from sensor failures. The system's ability to model temporal dependencies makes it well-suited for real-time fault detection in safety-critical applications.
