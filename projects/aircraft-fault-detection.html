Project Overview

This project implements a fault detection system for aircraft engines using Dynamic Bayesian Networks (DBN) integrated with Markov Random Fields (MRF). The system analyzes multivariate sensor data from simulated aircraft engine scenarios inspired by NASA's C-MAPSS framework, achieving 85.1% accuracy in fault classification.

The challenge was to distinguish between actual mechanical degradation (Oil Leak, Bearing Wear) and sensor failures (EGT Sensor Failure, Vibration Sensor Failure) while dealing with highly noisy vibration data typical in aircraft engines. The solution employs probabilistic graphical models with MRF smoothing to capture temporal dependencies and reduce false positives by 15%.

def discretize_sensor_data(data, method='quantile', n_bins=5):
    """
    Discretize continuous sensor readings into categorical bins

    Args:
        data: Sensor readings (numpy array)
        method: 'quantile', 'kmeans', or 'uniform'
        n_bins: Number of discrete bins (5 used in final model)
    """
    if method == 'quantile':
        bins = np.percentile(data, np.linspace(0, 100, n_bins+1))
        return np.digitize(data, bins[1:-1])

3. MRF Smoothing for Vibration Channels

To handle the inherent noise in vibration sensors, Markov Random Field smoothing enforces temporal consistency, reducing false positives by 15%: 

# MRF smoothing implementation
def apply_mrf_smoothing(predictions, lambda_mrf=0.7):
    """
    Apply MRF smoothing to enforce temporal consistency
    Reduced false positives by 15% in vibration channels

    Args:
        predictions: Raw DBN state predictions
        lambda_mrf: Smoothing strength (0.7 optimal)
    """
    smoothed = predictions.copy()
    for t in range(1, len(predictions)):
        # Penalize abrupt state transitions
        if predictions[t] != predictions[t-1]:
            # Apply temporal consistency constraint
            smoothed[t] = lambda_mrf * predictions[t] + \
                         (1 - lambda_mrf) * smoothed[t-1]
    return smoothed

Challenges and Solutions

Vibration Noise

Challenge: High levels of noise in vibration sensor readings made it difficult to distinguish between normal variations and actual bearing wear faults.

Solution: Implemented MRF smoothing with λ=0.7 to enforce temporal consistency, reducing false positives by 15% while maintaining detection sensitivity.

Sensor vs Mechanical Faults

Challenge: Distinguishing between actual mechanical degradation (oil leak, bearing wear) and sensor failures that mimic fault signatures.

Solution: Designed DBN structure to model sensor consistency across related channels (e.g., oil pressure affects temperature), achieving 85.1% accuracy across all 5 scenarios.

Key Insights

  • Quantile binning (5 bins) outperformed k-means and uniform discretization strategies
  • MRF smoothing with λ=0.7 optimal for vibration channels (15% false positive reduction)
  • Accuracy-tuned threshold outperformed macro-F1 tuned threshold by 3.2%
  • DBN temporal modeling captured progressive degradation in oil leak and bearing wear
  • 9-sensor subset (selected via correlation analysis) matched full sensor performance

Conclusion

This project demonstrates the effectiveness of probabilistic graphical models in handling complex, noisy sensor data for fault detection. The combination of Dynamic Bayesian Networks with MRF smoothing achieved 85.1% accuracy across 5 simulated scenarios, successfully distinguishing mechanical faults from sensor failures.

The macro-F1 score of 0.796 shows balanced performance across all scenarios, with MRF smoothing reducing false positives by 15% in noisy vibration channels. The system's ability to model temporal dependencies makes it well-suited for real-time fault detection in safety-critical applications.
