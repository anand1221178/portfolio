Overview

A sophisticated 2D physics simulation that models a drone-based payload drop system. The simulation features a drone flying at configurable altitude that must release a small ball bearing (5g, 8mm diameter) to accurately hit a ground target.

The system implements realistic ballistic physics including gravity, quadratic air resistance (drag), and wind effects, with an intelligent targeting system that calculates optimal release points using binary search.

Key Features

RK4 Integration

Runge-Kutta 4th order numerical integration using the GNU Scientific Library for accurate trajectory simulation.

Realistic Physics

Quadratic drag model, configurable air density, wind forces, and terminal velocity calculations.

Smart Targeting

Binary search algorithm finds optimal release point with 1cm accuracy tolerance.

Real-time Visualization

SDL2-based renderer at 1280x720 with trajectory prediction and statistics overlay.

Physics Implementation

Drag Force Model

The simulation uses the standard quadratic drag equation for spherical objects: 

// Drag force calculation
F_drag = 0.5 * rho * Cd * A * v^2

// Where:
// rho = 1.225 kg/m^3 (air density at sea level)
// Cd  = 0.47 (sphere drag coefficient)
// A   = pi * r^2 (cross-sectional area)
// v   = velocity magnitude

Numerical Integration

The RK4 integrator provides O(dt^4) global error compared to O(dt) for Euler integration. This is critical for accurate long-range trajectory prediction. 

// RK4 integration step
void rk4_step(state *s, double dt, physics_config *cfg) {
    vec3 k1_v, k1_p, k2_v, k2_p, k3_v, k3_p, k4_v, k4_p;
    
    // k1 = f(t, y)
    k1_v = compute_acceleration(s->pos, s->vel, cfg);
    k1_p = s->vel;
    
    // k2 = f(t + dt/2, y + k1*dt/2)
    // ... and so on for k3, k4
    
    // y_new = y + (dt/6) * (k1 + 2*k2 + 2*k3 + k4)
}

Targeting Algorithm

The targeting system uses binary search to find the optimal release point. It simulates the projectile trajectory for candidate release points and converges on the position that results in a hit within 1cm tolerance.

Key Insight

The relationship between release position and impact point is monotonic (earlier release = further impact), making binary search ideal. The algorithm converges in O(log n) iterations where n is the precision requirement.
