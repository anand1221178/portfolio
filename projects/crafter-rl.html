Project Overview

This project tackles the sparse reward problem in the Crafter environment by implementing a Proximal Policy Optimization (PPO) agent augmented with an Intrinsic Curiosity Module (ICM). The ICM encourages exploration by generating an intrinsic reward based on the agent's ability to predict the consequences of its actions, which is crucial in an environment where extrinsic rewards are rare.

Through a series of 11 experiments, the final PPO+ICM agent achieved a score of 8.61%, a 69.5% improvement over the baseline PPO agent. This demonstrates the effectiveness of intrinsic motivation for exploration in complex, sparse-reward environments like Crafter.

Key Features

Intrinsic Curiosity Module

A forward model predicts the next state's features, and the prediction error is used as an intrinsic reward.

Feature Encoding

An inverse model learns to predict the action taken between two states, forcing the feature encoder to learn state features that are relevant to the agent's actions.

Dual Reward System

The agent is trained on a combination of the extrinsic reward from the environment and the intrinsic reward from the ICM.

Iterative Experimentation

The final result was achieved through a series of 11 experiments, which included 7 documented failures that provided valuable insights.

Conclusion

This project demonstrates that intrinsic curiosity is a powerful tool for exploration in sparse-reward environments. The final PPO+ICM agent achieved a 69.5% improvement in score over the baseline, showcasing the effectiveness of the approach. The project also highlights the importance of hyperparameter tuning, as the balance between intrinsic and extrinsic rewards was found to be a critical factor for successful learning.
